Danduk Skanda – Laravel School Management Platform

View Repository

This system is designed to simplify the management of academic data such as classes, majors, students, attendance, subject teachers, and homeroom teachers. All features are built using Laravel 10 and utilize complex database relationships to ensure data consistency.

Danduk Skanda consists of several core modules: class management, student data, homeroom teachers, attendance, subjects, and academic year. Each feature uses Laravel's MVC architecture with server-side validation, Eloquent relationships, and a UI built using Bootstrap 5.

  • CRUD Kelas, Siswa, Jurusan, dan Mata Pelajaran
  • Manajemen Wali Kelas & Guru Pengajar
  • Sistem Presensi siswa
  • Import data siswa via Excel
  • Relasi database kompleks menggunakan Eloquent ORM
  • Blade templating + Bootstrap untuk UI

Focused on performance, data consistency, and ease of use. I ensured that each module was efficiently interconnected using hasMany, belongsTo, and many-to-many relationships when necessary. Additionally, an Excel import feature was added to speed up large data input.
