This project is a fullstack job portal system designed to simulate real-world recruitment processes. The frontend is built as a React SPA, while the backend provides a RESTful API using Laravel 12.

  • Designed a clear separation between frontend and backend using REST API architecture.
  • Implemented role-based access control for Admin, Employer, and Applicant.
  • Built secure authentication flow between React frontend and Laravel API.
  • Handled dynamic job listings, applications, and status tracking.
  • Maintained consistent UX across different user roles.

  • Developed RESTful API endpoints with protected routes using auth middleware.
  • Implemented role-based authorization aligned with frontend access control.
  • Integrated React frontend with API for job listings, applications, and user actions.
  • Designed scalable API structure to support future feature expansion.
  • Used clean and reusable components to maintain frontend consistency.
