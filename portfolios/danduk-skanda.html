Danduk Skanda — Laravel School Management Platform

View Repository

This system is designed to simplify the management of academic data such as classes, majors, students, attendance, subject teachers, and homeroom teachers. All features are built using Laravel 10 and utilize complex database relationships to ensure data consistency.

Danduk Skanda consists of several core modules: class management, student data, homeroom teachers, attendance, subjects, and academic year. Each feature uses Laravel's MVC architecture with server-side validation, Eloquent relationships, and a UI built using Bootstrap 5.

  • Designing accurate relational structures for interconnected academic entities
  • Preventing data duplication when importing large Excel datasets
  • Ensuring consistent attendance logic across homeroom teachers and subjects
  • Creating role-based access logic without breaking existing modules
  • Maintaining UI simplicity while handling complex academic workflows

  • Structured database design using one-to-many and many-to-many Eloquent relationships
  • Excel import system equipped with validation & duplicate-detection rules
  • A unified attendance engine with reusable functions for future modules
  • Role-based authorization using Laravel Gates & Policies
  • Component-based Blade templates for cleaner and more maintainable UI
  • Optimized queries using eager loading to improve dashboard performance

Key Features

  • Class & Student Management
  • Major & Subject Management
  • Homeroom & Teacher Assignment
  • Student Attendance System
  • Excel Data Import
  • Eloquent ORM Relationships
