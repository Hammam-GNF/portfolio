Growia E-Learning QA Project

Growia E-Learning Platform Testing

growia.education

As part of the QA team, I conducted manual testing for Growia’s e-learning web platform. The project focused on identifying usability issues, ensuring core learning features ran smoothly, and validating performance consistency across user roles (admin & buyer).

Created and executed structured test cases to validate authentication, content management, user enrollment, and attendance modules. Conducted cross-browser checks and documented outcomes through comprehensive test reports.

Several critical bugs were identified, including a spinner freeze during login, SQL injection vulnerability, and incorrect session handling. Each issue was logged, replicated, and retested after fix deployment.

Out of 74 total test cases, 46 passed, 17 failed, and 11 were pending. Critical bugs were resolved before the project handoff, ensuring a stable release and improved user experience for instructors and learners alike.
